#### 誠實評估一下

CKS 是**純實作考試**，考的不是選擇題，而是直接在真實叢集上操作。所以「不看影片、直接練習」這個策略能不能成功，完全取決於你現在的起點。

**問自己幾個問題：**

1. 你有 CKA 嗎？（CKS 的報考前提是先通過 CKA）
2. 平常工作有在用 Kubernetes 嗎？熟悉到什麼程度？
3. 對 Linux 基礎熟嗎？（AppArmor、Seccomp、capabilities 都是 Linux 層的東西）
4. 對資安概念有基礎嗎？還是完全陌生？

------

#### 三種情境

**情境 A：有 CKA、平常在用 K8s** → 完全可以跳過影片，直接讀文件 + 練習。影片對你來說只是浪費時間。

**情境 B：有 CKA、但 K8s 用得不深** → 建議至少把概念文件讀熟（像剛才那份講義），搞懂每個主題的「為什麼」，再進入練習。純粹硬練但概念模糊，遇到變化題會卡住。

**情境 C：CKA 剛過、對安全概念陌生** → 直接練習會很痛苦，因為你會一直不知道自己在幹嘛、為什麼這樣做。這種情況影片或文件二選一，至少要有一個。

#### 一個實際的建議

與其問「能不能不看影片」，不如問「我哪些主題還不懂」。
