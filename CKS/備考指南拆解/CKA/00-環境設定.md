# CKA 環境設定

## 本機練習 Cluster

```bash
# 用 kind 快速建立 k8s 1.35 cluster
bash scripts/init-cluster.sh

# 重建（清除後重來）
bash scripts/init-cluster.sh --recreate
```

## 考試環境準備（第一分鐘必做）

```bash
# alias 與 vim 設定
source scripts/exam-setup.sh
```

常用 alias：
```bash
alias k=kubectl
alias kn='kubectl config set-context --current --namespace'
export do='--dry-run=client -o yaml'
export now='--force --grace-period=0'
```

> ⚠️ 考試每道題 SSH 連線不共用 alias，只有 `k` 是預設設好的。
> 練習時可用 alias，考前 1-2 週改練原始指令。

## vim 設定

```bash
cat >> ~/.vimrc <<EOF
set expandtab
set tabstop=2
set shiftwidth=2
EOF
```

## 考試環境注意事項

- 終端輸入有 50-100ms 延遲，Tab 補全也會慢
- 可開兩個 terminal tab：一個做題、一個驗證
- 瀏覽器只能開 kubernetes.io（無書籤，用搜尋列）
- 複製貼上：`Ctrl+Shift+C` / `Ctrl+Shift+V`

## 練習 Cluster 選擇

| 主題 | 環境 |
|------|------|
| CNI 安裝、kubeadm、etcd、kubelet | Vagrant/VM |
| RBAC、NetworkPolicy、PSA、Falco、Istio | k3s 或 kind |
| PV/PVC、Ingress、Pod 設定 | 兩者皆可 |

## Vagrant + VirtualBox 快速指令

```bash
# 三台同時 snapshot（cluster 建好後先存）
vagrant snapshot save controlplane baseline
vagrant snapshot save worker1 baseline
vagrant snapshot save worker2 baseline

# restore（三台一起）
vagrant snapshot restore controlplane baseline
vagrant snapshot restore worker1 baseline
vagrant snapshot restore worker2 baseline

# 暫停保留狀態
vagrant suspend
vagrant up
```
