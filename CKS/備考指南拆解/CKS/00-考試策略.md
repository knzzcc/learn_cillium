## 常見失分陷阱

- **忘記切換 context**：每道題第一件事 `k config use-context <name>`
- **namespace 搞錯**：題目有指定就設好 `kn <ns>`
- **NetworkPolicy 忘記開 DNS egress（UDP 53）**：最常見錯誤
- **AppArmor 用舊 annotation 寫法**：v1.34 要用 `securityContext.appArmorProfile`
- **Audit logging 忘記 volume mount**：flag 加了但目錄沒掛，API server 起不來
- **EncryptionConfig provider 順序錯**：`identity: {}` 必須放最後
- **Secret data 欄位忘記 base64**：`data` 要 base64，`stringData` 是明文
- **沒驗證結果**：做完一題記得 verify，別假設沒問題
