在 Linux terminal 中，這些快捷鍵會大幅提升速度：

**逐字移動**

| 快捷鍵                  | 效果         |
| ----------------------- | ------------ |
| `Ctrl + →` 或 `Alt + F` | 向右跳一個字 |
| `Ctrl + ←` 或 `Alt + B` | 向左跳一個字 |

**跳到行首/行尾**

| 快捷鍵     | 效果     |
| ---------- | -------- |
| `Ctrl + A` | 跳到行首 |
| `Ctrl + E` | 跳到行尾 |

**刪除（順便省去移動）**

| 快捷鍵     | 效果                     |
| ---------- | ------------------------ |
| `Ctrl + W` | 刪除游標左邊整個字       |
| `Alt + D`  | 刪除游標右邊整個字       |
| `Ctrl + U` | 刪除游標到行首的所有內容 |
| `Ctrl + K` | 刪除游標到行尾的所有內容 |

**其他實用鍵**

| 快捷鍵     | 效果                     |
| ---------- | ------------------------ |
| `Ctrl + L` | 清空畫面（等於 `clear`） |
| `Ctrl + R` | 搜尋歷史指令             |
| `Ctrl + C` | 中斷目前指令             |
